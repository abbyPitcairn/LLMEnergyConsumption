# LLM Energy Consumption - Experimental Research

The aim of this research is to discover the average energy cost of prompting an LLM for text-to-text generation. First, we develop a dataset representative of the average LLM prompt and then test the energy cost by running this set of prompts through a handful of different LLMs. Finally, these experimental results will be analyzed and a final estimate of energy cost will be given in Watts per hour. 

### Execution

To run the program:

* Download project files. 
* Run the requirements installation and Python commands below. 
* Now your dataset is generated at the output csv file.
  
```
pip install -r requirements.txt
python DatasetGenerator.py
```
